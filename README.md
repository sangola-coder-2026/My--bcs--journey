# Student Marks Manager 🎓

Python + JSON वापरून बनवलेला Student Management System.BCS/Placement साठी Perfect Project.

### ✨ Features
- Student add करणे + 3 subjects चे marks
- सर्व Students ची List with Percentage
- Class Topper शोधणे 
- 35% पेक्षा कमी marks असलेले Fail Students
- JSON File मध्ये Data Save - App बंद केला तरी data राहतो

### 🛠️ Technology Used
Python, Dictionary, File Handling, JSON

python student_manager.py
